# Startup Failures Analysis

Este projeto tem como objetivo **analisar dados de falhas de startups** em diferentes setores, buscando entender **padrões financeiros, setoriais e de mercado** que influenciam o sucesso ou fracasso das empresas.

## Estrutura do Projeto

startup-failures/
│
├── data/ # Pasta para armazenar datasets
├── notebooks/ # Notebooks de exploração e análise
├── scripts/ # Scripts Python de limpeza e análise
├── .gitignore
├── README.md
└── requirements.txt

## Tecnologias utilizadas

- Python 3.10+
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook
